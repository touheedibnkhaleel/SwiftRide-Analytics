# SwiftRide-Analytics

SwiftRide Analytics is an end-to-end web app that transforms raw trip data into actionable insights and real-time fare predictions. It features executive dashboards for revenue, fleet distribution, and city performance, along with operational analytics like demand heatmaps and surge pricing analysis.

The app includes a Random Forest-based ML model for fare prediction with high accuracy (R²: 0.9973, MAE: 13 PKR). Built using Python, Streamlit, SQLite, Scikit-Learn, Plotly, and Pandas, it demonstrates the complete pipeline from database design to deploying an interactive ML application.
